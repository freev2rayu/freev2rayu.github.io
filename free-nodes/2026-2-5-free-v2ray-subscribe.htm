3.3 Tor网络：洋葱路由的哲学

如同其洋葱图标暗示的，Tor通过至少3层随机节点（入口→中间→出口）构建匿名路径。每个节点只能解密一层信息，这种设计使得2013年斯诺登泄密事件中，记者能够安全接收机密文件。

现实悖论：虽然Tor提供最强匿名性，但其出口节点可能被恶意运营者监控，因此安全专家建议"Tor+VPN"的组合方案。

FAQ

Shadowrocket 节点测速功能如何优化游戏体验？
Shadowrocket 会定期测速节点延迟和丢包率，并结合策略组自动切换最佳节点。保证游戏流量走低延迟节点，减少延迟和掉线，提高游戏流畅度和稳定性。
如何在 V2Ray 中自定义路由规则？
在 V2Ray 中可以通过修改 config.json 文件中的 routing 部分来自定义路由规则。用户可以根据 IP 段、域名或地理位置来决定流量的去向，从而实现自动分流或访问加速的效果。
V2Ray 的路由规则如何实现按应用分流？
V2Ray 支持按用户标签或端口匹配应用流量，结合策略组可实现指定应用走特定节点。这样可以保证游戏、视频和浏览器流量各自走最优线路，提高访问效率。
Sing-box 的 JSON 配置文件常见语法错误如何快速定位？
通过 JSON 校验工具或编辑器（例如 VSCode）打开配置文件，查看语法高亮与错误提示；也可运行 sing-box 并观察控制台错误信息，错误通常会指出第几行或哪个字段不合法以便修正。
Trojan 的伪装域名如何正确配置？
伪装域名必须真实可访问且证书有效，否则客户端无法建立连接。正确配置可隐藏真实流量特征，降低被封锁和检测风险，提高节点安全性和长期可用性。
V2Ray 如何实现分流规则？
V2Ray 使用 routing 模块实现分流，可根据目标域名、IP、端口或协议类型决定流量走向。用户可自定义多个 outbound，并通过规则实现国内直连、国外代理等智能路由功能。
Hysteria2 多路复用如何提升短连接性能？
多路复用允许多个短连接共享同一通道，减少握手延迟和资源消耗。适合高频短连接场景，如游戏和即时通讯，可提高吞吐量和连接稳定性，优化整体网络体验。
Clash 支持多配置文件加载吗？
Clash 支持多配置切换，用户可在图形界面中导入多个 YAML 文件，并通过选择不同配置快速切换使用场景。例如家用模式、移动模式或公司网络模式。
Nekoray 节点订阅失败如何解决？
确认订阅 URL 是否有效并检查网络通畅。可手动刷新或重新添加订阅链接，保证节点列表最新。结合策略组使用，可确保稳定科学上网体验。
Clash Premium 与普通版本有什么区别？
Clash Premium 是增强版，支持更多功能如 TUN 模式、本地 DNS 解析、脚本规则和自定义代理链。它能实现更细致的流量控制和兼容更多协议，适合对网络有高级需求的用户使用。
